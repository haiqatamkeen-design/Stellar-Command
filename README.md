# Stellar Command — Spacecraft Maintenance System

A console-based spacecraft maintenance management system built in C++. It tracks spacecraft components, schedules maintenance tasks, and logs operational status, applying object-oriented programming principles for a clean, modular design.

## Features

- Add, view, and manage spacecraft components
- Schedule and track maintenance activities
- Log and update operational status of systems
- File-based data persistence so records are saved between sessions
- Menu-driven interface for easy navigation

## Tech Stack

- **Language:** C++
- **Concepts:** Object-oriented programming (classes, encapsulation), file handling

## How to Run

1. Make sure you have a C++ compiler installed (e.g., g++ or any IDE such as Code::Blocks or Dev-C++).
2. Compile the program:
   ```
   g++ main.cpp -o stellar_command
   ```
3. Run the program:
   ```
   ./stellar_command       # On Windows: stellar_command.exe
   ```
